# Cálculo de Distancia Más Cercana
Este programa en C++ permite:
- Ingresar una cantidad de puntos en el plano.
- Ingresar un punto de referencia.
- Calcular cuál punto es el más cercano al punto ingresado.
- Mostrar la distancia mínima y el punto correspondiente.
## Estructura
- struct Point
- Función calcularDistanciaMasCercana
- Función main con entrada por teclado
## Cómo compilar
g++ Parcial_1.cpp -o ejecutable
## Cómo ejecutar
./ejecutable
